# AI-Summarizer-extension
Instantly summarize any article using Google Gemini AI with support for brief, detailed,  bullet points, and auto-generated MCQ formats.

A lightweight Chrome Extension that **instantly summarizes any article** using Google's Gemini AI API. 
Extract key information from web articles in multiple formats: Brief summaries, Detailed analysis, 
Bullet points, and Auto-generated Multiple Choice Questions.

** Features **

- **AI-Powered Summarization** - Uses Google Gemini API
- **Multiple Summary Types**:
  - Brief (50 words)
  - Detailed (150 words)
  - Bullet Points (5-7 key points)
  - Multiple Choice Questions (auto-generated MCQs)
- **One-Click Summarization** - Works on any webpage
- **Copy to Clipboard** - Easy sharing and saving
- **Secure API Key Storage** - Chrome Sync Storage
- **Fast & Lightweight** - Minimal resource usage
- **Beautiful UI** - Modern gradient design with smooth animations

** Technologies Used**

- **Frontend:** HTML5, CSS3, JavaScript
- **APIs:** Google Gemini API (v1beta)
- **Browser APIs:** Chrome Extension Manifest V3, Content Scripts
- **Storage:** Chrome Sync Storage
- **Architecture:** Popup UI, Content Script, Background Service Worker
